This workbench runs a deterministic mock pipeline on bundled permit/claim JSON. Inspired by public Brain Co. posts; not affiliated.

What institutional buyers need

GovOS · permitting

Blueprint extraction → zoning checks → application-level confidence. Low confidence on bedroom windows or setbacks routes to a reviewer instead of silent auto-approve.

InsuranceOS · claims

Policy sections → MECE IF-THEN rules → exclusion checks. Rules version drift and pre-existing conditions must surface as explicit receipt fields, not log noise.

plan → run → doctor → report on bundled JSON. Receipt is the artifact; dashboard is the inspector. Independent of sessiongrep PR #19.
